# Install and upgrade

To install the 2D Tooling package, follow these steps:

1. Open the Package Manager by going to **Window** &gt; **Package Management** &gt; **Package Manager**.
2. Select the **Unity Registry** from the Package Manager's [navigation panel](https://docs.unity3d.com/6000.0/Documentation/Manual/upm-ui-nav.html).
3. Search for the **2D Tooling** package and select **Install**. 

Make sure that the compatible version of the package is installed in the project:

2D Tooling package version  | Unity Editor version
--|--
1.x.x |  6000.3
